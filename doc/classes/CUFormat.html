CUFormat

A group of static methods that can format numbers, percentages, amounts of currency, as well as dates and time using the local formatting rules associated with a locale and, for some methods, the language of a locale.

You can refer to this class by its alias, which is UFmt.

static percent (number $number, CULocale $inLocale) : OOPString

Formats a number as a percentage and according to the formatting rules used in the default or some other locale and returns the formatted string.

timeWithPattern()

static timeWithPattern (CTime $time, CTimeZone $timeZone, string $pattern, CULocale $inLocale) : OOPString

Formats a point in time as a string in a specified time zone according to a specified pattern and the formatting rules used in the default or some other locale and returns the formatted string.

The formatting patterns that you can use to format a point in time with this method are described in Date Format Patterns of the Unicode Technical Standard #35.

Parameters

$time CTime

The point in time to be formatted.
$timeZone CTimeZone

The time zone in which the components in the resulting string are to appear.
$pattern string

The formatting pattern.
$inLocale CULocale

OPTIONAL. Default is the application's default locale. The locale in which the point in time is to be formatted.

Returns

OOP String —

A string with the formatted point in time.
